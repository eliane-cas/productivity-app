how to use this app

I realised that when i felt overwhelmed about the amount of tasks I had to do in a day it helped to plan the amount of pomodoros I wanted to dedicate to each task.
Cutting them up into 25min chunks make it seem a lot more manageable.
So I created this minimalist web-app, integrating a to-do list with a pomodoro timer.

  1. 1. In the to-do list assign the number of pomodoros you want to dedicate to the task
  2. 2. The total counter will update with the total pomodoros
  3. 3. Use the 25min timer
  4. 4. Each time you finish a pomodoro it will automatically update in the counter
